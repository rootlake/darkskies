# Dark Skies of California — Class of 1967 Faculty Impact Award

A scrolling web gallery documenting the 2026 scouting trip for Pomfret School's Honors Astronomy dark-sky expedition, planned for March 2027.

## Live Site

Hosted via GitHub Pages: [rootlake.github.io/darkskies](https://rootlake.github.io/darkskies)

## What It Is

A single-page site built in vanilla HTML/CSS/JS. As you scroll, the background transitions from a sunset gradient to deep black, and stars emerge to mirror the Bortle light-pollution scale — the organizing idea behind the course.

**Sections**
1. **Thank You** — hero with opening context for Class of 1967 donors
2. **The Scout** — animated route map (Las Vegas → Death Valley → Alabama Hills) with scouting notes
3. **The Bortle Scale** — interactive scale bar, reference photography, and the science behind the expedition
4. **2027 Expedition** — full itinerary (Malibu → Joshua Tree → Alabama Hills → Death Valley → Mt. Wilson)
5. **The Gear** — smart telescopes, DSLRs, 360° camera, and Sky Quality Meters
6. **Photo Gallery** — polaroid-style gallery from the scouting trip
7. **Closing** — thank-you with Pomfret crest

## Features

- Star field canvas that increases in density as you scroll (Bortle gradient effect)
- Polaroid photos develop from darkness on scroll-in (5-second reveal)
- Click any polaroid to open a lightbox
- Hover on any polaroid for 3D tilt + dynamic light glare
- Animated SVG route map draws on scroll
- Fixed left-side dot navigation
